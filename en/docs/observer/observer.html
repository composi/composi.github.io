Observer

Create an Observer

@composi/observe is a class. As such, to use it you need to first create a new instance:

import { Observer } from '@composi/observer'

const observer = new Obsever()

This will create a new observer object. By itself its not very useful. To use it you'll need to set up a watcher and then send an event and data to it.
