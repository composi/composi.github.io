Hydrate

Reusing Server-Side Content

You have content that gets rendered on the server, but after the page loads, you want to bring it to life with Composi functionality. You'll often want server-rendered content for faster page loads and SEO. Composi supports hydration through its function hydrate.

Hydrating Content

First you need to import hydrate from @composi/core:

import { hydrate } from '@composi/core'

To hydrate content, pass hydrate a reference to the element and capture the result in a variable. hydrate takes the element and creates a virtual node from it. You then pass this virtual node to your render function as the final argument:

import { hydrate } from '@composi/core'

const serverList = hydrate('#server-list')

// Mount new component and hydrate existing element:
let list = render(<List data={someData}/>, 'section', serverList)
