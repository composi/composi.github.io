create-composi-app

Rather than the drudge of installing all the parts to create a Composi project, you can use create-composi-app to get up and going quickly.

create-composi-app creates a poject with sound structure with a task run that watches your files and rebuilds and reloads the browser when you make changes. It also provides Eslint and Prettier for formatting.

The setup for a new project is like this:


my-project-|
    src-|
        css-|
        |-css-reset.css
        |-styles.css
        images-|
          icons-|
            |-favicon-16x16.png
            |-favicon-32x32.png
        js-|
          components-|
          |-title.js
        |-app.js
    |-.bablerc
    |-.editorconfig
    |-.eslintrc.json
    |-gulpfile.js
    |-index.html
    |-jsconfig.json
    |-package.json
    |-README.md
    |-rollup.config.js

Included Modules

When you create a new project, it includes the following @composi modules ready for you to import if you need to:

  • @composi/fragment
  • @composi/datastore
  • @composi/observer
  • @composi/idb
  • @composi/get-type
  • @composi/merge-objects

To use these you just need to import them into your project:

import { h, render } from '@composi/core'
import { Fragment } from '@composi/fragment'
import { DataStore } from '@composi/datastore'
import { Observer } from '@composi/observer'
import { idb } from '@composi/idb'
import { getType } from '@composi/get-type'
import { mergeObject } from '@composi/merge-objects'

Read the documentation from the links above to learn how to use them.
