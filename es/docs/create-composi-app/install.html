Instalación

Para instalar Composi, primero deberá instalar Node . Si ya tiene instalado Node, abre la terminal y ejecuta:

npm install --save-dev @composi/create-composi-app

Si está en una Mac o Linux, es posible que tenga que ejecutar el comando anterior con sudo.

Después de la instalación, puedes verificar que todo está bien por ejecutar:

create-composi-app -h
// o:
create-composi-app --help

Esto imprimirá ayuda en la terminal. También puede obtener la versión actual de create-composi-app :

create-composi-app -v
// o:
create-composi-app --version

Crear un Proyecto

Ahora que tiene instalado @composi/create-composi-app, puedes usarlo para crear una nuevo proyecto para usar @composi/core. Para hacer esto, abre la terminal y entra create-composi-app seguido del nombre del proyecto. Esto creará un proyecto con ese nombre en el escritorio:

create-composi-app MiProyecto

Si desea un nombre de proyecto de varias palabras con espacios, deberá incluir el nombre entre comillas:

create-composi-app "Mi Proyecto"

Cuando creas un proyecto con nombre de varias palabras con espacios, Composi crea una carpeta con ese nombre. Sin embargo, debido a los requisitos de NPM, el nombre package.json se convierte a minúsculas y los espacios a guiones. Por lo tanto, el nombre "Mi Proyecto" aparecerá en el archivo package.json como "mi-proyecto". Pero la carpeta del proyecto seguiría siendo "Mi Proyecto".

Provée una Senda

Por defecto, create-composi-app coloca el nuevo proyecto en el escritorio. Sin embargo, puede decirle dónde desea ubicar el proyecto proporcionando un segundo argumento. Esto debe ser una ruta válida para el sistema operativo. También necesita ser citado. Crearemos nuestro proyecto en una carpeta especial en nuestro directorio principal:

create-composi-app "Panda Puff" "~/dev"

Lo anterior creará el proyecto "Panda Puff" en la carpeta dev. Si dev no existe, se creará. De lo contrario, se creará dentro de la carpeta sin cambiar otras cosas en la carpeta. Date cuenta que los archivos con el mismo nombre que los del nuevo proyecto serán reemplazados por los nuevos.

Instalar Dependencias

Después de crear un nuevo proyecto hay que instalar las dependencias. Para hacerlo, cd a la carpeta del proyecto y ejecuta lo siguiente en la terminal:

npm i

Esto puede llevar un tiempo, ya que el valor predeterminado viene con una cantidad de complementos de procesamiento de imágenes que son grandes. La ejecución de imágenes requiere el código C++ para ejecutarse de manera eficiente, y esas bibliotecas deben descargarse y construirse antes de que puedan usarse. Una vez que la instalación haya finalizado, estará listo para iniciar su proyecto.

Lanza el Proyecto

Lanza el proyecto ejecutando el siguiente comando en la terminal mientras estás en la carpeta del proyecto:

npm start

Esto construirá el proyecto, configurará a los observadores y lanzará la compilación en el navegador predeterminado.

Cualquier cambio que realice en el archivo index.html, o cualquier archivo dentro de la carpeta src del proyecto, activará un proceso de compilación y la actualización del navegador.
