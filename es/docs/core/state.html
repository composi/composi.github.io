Estado

Gestión de Estado para los Componentes Funcionales

Los componentes de clase habilitan el estado local. Sin embargo, @composi/core es una librería funcional. Por su naturaleza, los componentes funcionales son no tiene estado. Seguramente se encontrarás necesitando un estado local para sus componentes funcionales. @composi/core proporciona un medio de gestión de estado para componentes funcionales a través de su entorno de ejecución atravéz de la función run. Esta es una implementación simplificada de TEA: La Architectura Elm.

SVU

Cada program del entorno de ejecución divide su funcionalidad entre tres métodos:

  • State: cualquier tipo válido de JavaScript
  • View: la renderización visual del estado del programa
  • Update: las acciones que devuelve una nueva versión del estado

Debido a que la gestión de estado es un sujeto tan complicado y que el entorno de ejecución tiene tantas características, tenemos una sección dedicada a ello: Entorno de Ejecución
